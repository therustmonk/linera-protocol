fn into_make_service(self) -> IntoMakeService<Self>

Convert this service into a MakeService, that is a Service whose response is another service.

This is commonly used when applying middleware around an entire Router. See “Rewriting request URI in middleware” for more details.

fn into_make_service_with_connect_info<C>(
    self
) -> IntoMakeServiceWithConnectInfo<Self, C>

Convert this service into a MakeService, that will store C’s associated ConnectInfo in a request extension such that ConnectInfo can extract it.

This enables extracting things like the client’s remote address. This is commonly used when applying middleware around an entire Router. See “Rewriting request URI in middleware” for more details.
