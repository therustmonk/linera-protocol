Module linera_views::common

source ·
Expand description

The definitions used for the KeyValueStoreClient and Context. This provides several functionalities for the handling of data. The most important traits are:

  • KeyValueStoreClient which manages the access to a database and is clonable. It has a minimal interface
  • Context which provides access to a database plus a base_key and some extra type E which is carried along and has no impact on the running of the system. There is also a bunch of other helper functions.
