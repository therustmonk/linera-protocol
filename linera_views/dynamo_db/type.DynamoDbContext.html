Creates a new DynamoDbContext instance using a LocalStack endpoint.

Requires a LOCALSTACK_ENDPOINT environment variable with the endpoint address to connect to the LocalStack instance. Creates the table if it doesn’t exist yet, reporting a TableStatus to indicate if the table was created or if it already exists.
